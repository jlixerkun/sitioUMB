Multiplicando Multiplicador Resultado

Una persona invierte en una cuenta de ahorros en un banco $1000.000 con un 6% de intereses anual, realice un programa que calcule el monto acumulado de la cuenta en un periodo de 5 años.

//Ejercicio 5.4
//Acumulado por interés 
$('.ej54').click(function (e) {
    var inversion = parseInt(prompt('Ingrese monto de inversión inicial en pesos', '1000000'))
      , periodo = parseInt(prompt('Ingrese término del ahorro en años', '5'))
      , interes = parseFloat(prompt('Ingrese interés anual en unidades porcentuales', '6.0'))
      , acumulado = inversion;

    for (i = 1; i <= periodo; i++) {
      acumulado += inversion * (interes / 100);
    }
    alert('Al cabo de ' + periodo + ' años, El acumulado es $' + acumulado);
    e.preventDefault();
  });

Ejercicio 7

Do while

  1. Crear un programa en JavaScript que permita mediante do-while mostrar por pantalla los números pares del 0 al 50

    //Ejercicio 71
  //Do While
  $('.ej71').click(function (e) {
    var numTope = parseInt(prompt('Ingrese el tope', '50'))
      , i = 0
      , text = '';
    do {
      text += i + ' ';
      i += 2;
    }
    while (i <= numTope);
    alert(text);
    e.preventDefault();
  });
